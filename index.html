Typography

Typography has been made to be scalable within' the limits of viewport and not to jump around. This is made possible by using vh and vw units and basing them in to px units

Theming

Starting your project is fast as the project takes advantage of var() & calc() css functions. It also includes easy colors configuration for fast development in mind.

units & system

SuperBro takes full advantage of EM, VH and VW units! This way it can be scalable through the fonts and let browser do so called calculations. Fastens development process.

Flexibility

The aim is to keep it minimal but configuratable as possible. Having best of both worlds is not easy. So if you decide to go with predefined values you can trust that I have configurated best possible line-heights and font-size for the project.

Support

These are the basic tags that are being supported. There will be more in future updates. You can suggest which ones would be perfect for this project. Or if you have new ways to implement some tags. Of course we won't include everything as the project's goal is to stay minimal (but configuratable) as possible.

Mobile / Media queries

Grid itself is really flexible and mobile friendly. In this project we are taking advantage of both, Grid layouts but also the media-queries. This might be updated later on and plan is to go without any media-queries, but for now we are going with fixed amount of grid-template-columns + media-queries to adjust them.
